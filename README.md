## Requirements
- [Pygame](https://pypi.org/project/pygame/)

## Usage
1. Install the required Python package:

   ```bash
   pip install pygame
   ```

2. Run the script:

   ```bash
   main.py
   ```

## File Structure
- `assets`: Folder with png images
- `main.py`: Python script for running the game
